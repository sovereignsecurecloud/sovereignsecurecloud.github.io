# Observability & Monitoring

Willkommen bei der Observability-Dokumentation der SovereignSecure Cloud. Ein tiefer Einblick in die Infrastruktur- und Anwendungsleistung ist für die Aufrechterhaltung von Hochverfügbarkeit und Sicherheit von entscheidender Bedeutung.

## Der Observability-Stack

Unsere Cloud nutzt einen modernen Open-Source-Observability-Stack, der Metriken, Protokolle (Logs) und Traces in der gesamten Umgebung erfasst:

* **Prometheus & Grafana:** Die Kern-Engine für die Erfassung von Zeitreihenmetriken, Alarmierung und visuelle Dashboards.

## Sichtbarkeit für Mandanten (Tenants)

Über die Cloud Management Platform (CMP) können Mandanten die Ressourcenauslastung (CPU, Speicher, Netzwerk) für ihre spezifischen Projekte anzeigen. Fortgeschrittene Benutzer können auch benutzerdefinierte Anwendungsmetriken für die Prometheus-Endpunkte der Cloud verfügbar machen oder isolierte Überwachungs-Stacks in ihren eigenen Kubernetes-Clustern bereitstellen.
