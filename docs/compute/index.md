# Compute Overview

Welcome to the SovereignSecure Cloud Compute documentation. The Compute service provides scalable, secure, and highly available virtualized infrastructure to run your applications and workloads. 

Powered by OpenStack Nova and centrally managed through our unified Cloud Management Platform (CMP), you can provision, monitor, and manage computing resources on demand.

## Core Compute Services

This section provides detailed guides on how to effectively utilize the various compute resources and functionalities available within SovereignSecure Cloud.

*   **[Instances](instance_user.md):** The standard virtual machines (VMs) that provide CPU, memory, and root storage. Learn how to provision, manage, and troubleshoot your instances.
*   **[GPU Instances](gpu_instance.md):** Specialized hardware-accelerated instances equipped with vGPUs or PCI-passthrough devices, designed for compute-intensive workloads such as Machine Learning, AI training, and HPC.
*   **[Images](image.md):** Pre-configured operating system templates used to launch instances quickly. Learn about public, private, and custom images, as well as supported formats like QCOW2.
*   **[Key Pairs](keypair.md):** Secure SSH key pairs used for authentication when connecting to your Linux instances. Understand how to create, import, and manage them.
